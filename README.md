# Кофейня — Сайт кофейни

## Описание
Сайт кофейни, созданный с использованием семантической HTML-разметки. Проект демонстрирует правильное использование HTML5 тегов для создания структурированной и доступной веб-страницы.

**Практическое занятие 5:** Реализован мобильный интерфейс с гамбургер-меню, адаптивной навигацией и кнопкой «Наверх».

## Используемые технологии
- HTML5 (семантическая разметка)
- CSS3 (BEM, Flexbox, Grid, медиазапросы)
- JavaScript (гамбургер-меню, scroll-to-top)
- Git & GitHub
- GitHub Pages

## Структура проекта
```
frontendbackend-practice5/
├── index.html          # Главная страница
├── styles/
│   ├── style.css      # Основные стили
│   └── mobile.css     # Мобильные стили (≤1024px)
├── scripts/
│   └── menu.js        # Гамбургер-меню и кнопка «Наверх»
├── assets/
│   ├── img/           # Изображения
│   ├── icons/         # Иконки и favicon
│   └── fonts/         # Шрифты (для будущего использования)
└── README.md          # Описание проекта
```

## Особенности (ПЗ 5)
- Адаптивное меню Desktop → Tablet → Mobile
- Гамбургер-меню на планшетах и мобильных (≤768px)
- Кнопка «Наверх» с плавной прокруткой
- Touch-зоны ≥44px (HIG/Material Design)
- Фиксация header при скролле
- Шрифты: 16px (MD) / 17px (iOS)
- Семантическая HTML-разметка
- Валидный HTML5 код
