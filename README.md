# Кофейня — Сайт кофейни

## Описание
Сайт кофейни, созданный с использованием семантической HTML-разметки. Проект демонстрирует правильное использование HTML5 тегов для создания структурированной и доступной веб-страницы.

## Используемые технологии
- HTML5 (семантическая разметка)
- Git & GitHub
- GitHub Pages

## Структура проекта
```
frontendbackend/
├── index.html          # Главная страница
├── assets/
│   ├── img/           # Изображения
│   ├── icons/         # Иконки и favicon
│   └── fonts/         # Шрифты (для будущего использования)
└── README.md          # Описание проекта
```

## Ссылки
- [Опубликованный сайт на GitHub Pages](https://Kusatka.github.io/frontendbackendmirea/)

## Особенности
- Семантическая HTML-разметка (header, nav, main, section, article, footer)
- Корректная структура заголовков (один h1 на странице)
- Валидный HTML5 код
- Адаптивная верстка (viewport meta tag)
- Правильная типографика
